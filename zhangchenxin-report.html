① 语法优先突破

虚拟语气（wish + had done / would not…）、时态序列（过去完成时）、定语从句（where/whose）、倒装句（not until…）是你本次失分最集中的四个语法项，也是留学英语的高频考点。建议每个语法板块专项训练，最少完成20道配套练习，重点理解"时间轴与事实的对应关系"。

② 词汇分类扩展

重点补充学科类词汇：医学健康（allergic, fatigue）、自然地理（marine, carnivorous）、宗教建筑（cathedral）。建议用主题词汇卡，按类成组记忆，同时注意同义近义词的精确辨析（如 church vs cathedral / memory vs memoirs）。

③ 连接词与固定搭配强化

阅读句子填空的失分源于 cause of / reason for 的搭配混淆，建议专项整理因果类连接词的搭配规则（because of / due to / result in / cause / reason for），通过仿写练习强化记忆。

④ 保持阅读与听力优势

阅读和听力是你明显的竞争优势，建议每天坚持精读一篇英文文章并注意连接词用法，同时保持英文听力输入（播客/纪录片），进阶目标是在细节题和信息捕捉上进一步提升准确率，争取两项满分。
